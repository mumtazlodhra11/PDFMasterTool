========================================
COMMON ISSUES & FIXES:
========================================

ISSUE 1: "ERR_CONNECTION_REFUSED"
---------------------------------
Fix:
- Server restart karein
- Port check: netstat -ano | findstr :9001
- Firewall check karein

ISSUE 2: Page blank load ho raha hai
------------------------------------
Fix:
- Browser console check (F12)
- Clear cache: .\clear-cache.ps1
- Hard refresh: Ctrl + Shift + R

ISSUE 3: "before-hydration.js" error
------------------------------------
Fix:
- Cache clear karein
- Server restart
- Check astro.config.mjs has exclude: ['@astrojs/react']

ISSUE 4: Port 9001 already in use
---------------------------------
Fix:
```powershell
netstat -ano | findstr :9001
taskkill /PID <PID> /F
npm run dev
```
