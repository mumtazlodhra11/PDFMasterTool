Are my files uploaded to servers?

Most tools run fully in your browser using client-side processing. Your files never leave your device, ensuring complete privacy. Only heavy conversions that require more processing power use our secure Cloud Run backend temporarily, and files are deleted immediately after processing. We never store your files.

How secure is PDFEliteTools?

PDFEliteTools uses HTTPS encryption for all connections. Client-side tools process files entirely in your browser, so your data never leaves your device. For cloud-based processing, files are encrypted in transit and deleted immediately after processing. We don't track your usage or store any personal information.

Do you collect or store my personal information?

No. We don't require registration, login, or any personal information. We don't track your usage, store your files, or collect any data about you. Your privacy is our top priority.

Are there watermarks or sign-up required?

No watermarks ever. No account or sign-up needed. All tools are completely free to use without any limitations or restrictions. You can use all features without creating an account.

What file sizes are supported?

Most tools support up to 150MB files, which is higher than most competitors (typically 20-50MB). For very large files, we recommend using the compress tool first to reduce file size before processing.

How fast are the conversions?

Client-side tools are instant - they process files directly in your browser. Backend conversions are optimized and typically complete in seconds, depending on file size and complexity. Most operations finish within 10-30 seconds.

Which browsers are supported?

PDFEliteTools works on all modern browsers including Chrome, Firefox, Edge, Safari, and Opera. We recommend using the latest version of your browser for the best experience.

Can I use PDFEliteTools on mobile devices?

Yes! PDFEliteTools is fully responsive and works on mobile devices, tablets, and desktops. Some tools like Mobile Scanner are specifically designed for mobile use. However, for best results with large files, we recommend using a desktop browser.

What should I do if a tool isn't working?

First, try refreshing the page and clearing your browser cache. Make sure you're using a supported browser and that your file meets the size requirements. If the issue persists, check our Support Center or contact us for assistance.

Can I process password-protected PDFs?

Some tools can work with password-protected PDFs if you enter the password when prompted. However, certain operations may require the PDF to be unlocked first. Use our Unlock PDF tool if you need to remove password protection.

Why is my file taking longer to process?

Processing time depends on file size, complexity, and the type of operation. Large files, files with many pages, or complex operations like OCR may take longer. If processing seems stuck, try refreshing the page and uploading the file again.
